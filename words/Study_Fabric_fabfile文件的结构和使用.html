Study_Fabric_fabfile文件的结构和使用

Posted April 07, 2017

1.指定fabfile

在命令行中通过 -f :fab -f fabtasks.py
在 fabricrc 中指定:在 ~/.fabricrc 中添加 fabfile = fabtasks.py;
如果fabfile不在当前目录,支持使用相对路径和绝对路径表示以及~家目录表示方法

2.引用fabric

Fabric 本质上依然是 Python，因此你 可以 随意地调用它的组件。不过，处于封装和便捷性（以及 Fabric 脚本的易用性）考虑，Fabric 的公开 API 由 fabric.api 模块维护


Fabric 的 Operations 、上下文管理器 、 装饰器 以及 实用工具 都是本模块的名字空间，为 fabfile 提供了一套简单并且统一的接口;

Bash
from fabric.api import *
    等同于:
    from fabric.api import abort, cd, env, get, hide, hosts, local, prompt, \
    put, require, roles, run, runs_once, settings, show, sudo, warn
