Django学习笔记01_web框架基础

Posted April 07, 2017

学习DJango前需要了解的框架基础知识

1.所有的web框架都是基于socket

eg01:简单socket程序

Python
# -*- coding: utf-8 -*-
'''
File Name: test_socket.py
Author: WillDX
mail: xiang.dai@shuyun.com
'''
import socket

def handle_request(client):
    buf = client.recv(1024)
    client.send("HTTP/1.1 200 OK\r\n\r\n")
    client.send("Hello, daixiang!")

def main():
    sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    sock.bind(('localhost',8000))
    sock.listen(5)
    while True:
        connection, address = sock.accept()
        handle_request(connection)
        connection.close()

if __name__=="__main__":
    main()
    print "Serving HTTP on port 8000...."

    """web app"""

3. 使用wsgiref封装好的web server部分，自定义web app部分扩展功能

eg03:扩展不同的URL返回不同的网页内容

Python
# -*- coding: utf-8 -*-
'''
File Name: extentd_wsgiref.py
Author: WillDX
mail: xiang.dai@shuyun.com
'''
from wsgiref.simple_server import make_server

def login():
    return '<h1>Login</h1>'
def logout():
    return '<h1>Logput</h1>'
def index():
    return '<h1>Index</h1>'

url = [
    ("/login/", login),
    ("/logout/", logout),
    ("/index/", index),
]

def RunServer(environ, start_response):
    start_response('200 OK',[('Content-Type', 'text/html')])
    user_path = environ["PATH_INFO"]
    serv_path = None
    for i in url:
        if i[0] == user_path:
            serv_path = i[0]
            return i[1]()
    if not serv_path:
        return "页面没有找到!"·

if __name__=="__main__":
    httpd = make_server('', 8000, RunServer)
    print "Serving HTTP on port 8000...."
    httpd.serve_forever()

5.MVC/MTV框架

Raw
mvc:模型，视图，控制端 
    如果我们使用MVC模式的话我们一般将核心处理代码放在控制端，例如建立一个控制端文件夹专们用来放置处理逻辑代码；将前端显示界面比如说html文件放在视图文件夹；简单来说，只是一个约定俗成的规则；
mtv:模型，模板，视图
    另外一种分类摆放代码的方式，将处理代码放到view，将前端页面放在模板；Django是MTV模式的；
