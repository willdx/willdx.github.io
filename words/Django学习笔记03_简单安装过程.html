Django的安装过程

1.配置

Python
mkvirtualenv  willdjango   #创建虚拟环境
    pip install Django==1.8      #安装Django
    django-admin startproject willdjango  #开启项目
    cd willdjango   #进入项目目录
    python manage.py startapp app  #创建APP
    #manage.py文件是Django程序的入口文件
    python manage.py runserver 0.0.0.0:8000  #启动服务

2.浏览器访问http://127.0.0.1:8000/

Raw
出现 It worked! 表示成功

3.简单URL请求对应views函数执行

Python
`vim willdjango/urls.py`
    from django.conf.urls import url
    from django.contrib import admin
    from app.views import index  #添加

    urlpatterns = [
        url(r'^admin/', admin.site.urls),
        url(r'^index/', index),  #添加
    ]

    `vim app/views.py`
    from django.shortcuts import render
    from django.http.response import HttpResponse
    
    def index(request):
        return HttpResponse("index")

runserver

再次runserver,然后浏览器访问：http://127.0.0.1:8000/index/，返回index，表示成功
