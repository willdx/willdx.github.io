1.基本使用

Bash
$fab task1 task2

2.运行任意远程shell命令

Bash
$ fab [options] -- [shell command]
例如: fab -H system1,system2,system3 -- uname -a

4.每个任务的参数

使用冒号（：）任务名称从它的参数分开;
使用逗号（，）将参数彼此分离（可通过使用反斜线，转义即\）;
使用等号（=）的关键字参数，或者忽略他们的位置参数。也有可能是转义反斜杠。

Raw
实例代码：
Bash
def new_user(username, admin='no', comment="No comment provided"):
        print("New User (%s): %s" % (username, comment))
        pass
    run:
        fab new_user:myusername,admin=no,comment='Gary\, new developer (starts Monday)'

5.角色和主机

Bash
$ fab new_user:myusername,hosts="host1;host2"
