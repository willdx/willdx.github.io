1.运行命令,生成doc工程

Bash
sphinx-quickstart

2.美化主题

Python
pip install sphinx_rtd_theme
Python
vim docs_bak/source/conf.py
将原有的: html_theme = 'alabaster' 注释掉
然后添加如下行到任意位置:
import sphinx_rtd_theme
html_theme = 'sphinx_rtd_theme'
html_theme_path = [sphinx_rtd_theme.get_html_theme_path()]
html_title = "Godbox Documentation"
html_short_title = ""
html_show_copyright = False

3.生成rst

Python
# 进入setup.py文件目录,也就是项目根目录
sphinx-apidoc -o docs/source godbox

4.生成html

Python
cd docs
make html
