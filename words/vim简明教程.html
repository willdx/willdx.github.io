简明VIM

Posted April 07, 2017

第一级 存活

iInsert 模式，按 ESC 回到 Normal 模式.


x删当前光标所在的一个字符。


:wq 存盘 + 退出 (:w 存盘, :q 退出) 注：:w 后可以跟文件名


dd 删除当前行，并把删除的行存到剪贴板里


p粘贴剪贴板

推荐: 1. 使用键盘hjkl移动光标 2. :help <command>显示相关命令的帮助。你也可以就输入 :help 而不跟命令。退出帮助需要输入:q

第二级 感觉良好

1. 各种插入模式

a → 在光标后插入


o 在当前行后插入一个新行


O在当前行前插入一个新行


cw替换从光标所在位置后到一个单词结尾的字符

2. 简单的移动光标

0 → 数字零，到行头


^ 到本行第一个不是blank字符的位置 注所谓blank字符就是空格，tab，换行，回车等


$到本行行尾


g_到本行最后一个不是blank字符的位置


/pattern 搜索 pattern 的字符串 注：如果搜索出多个匹配，可按n键到下一个

3. 拷贝/粘贴 注：p/P都可以，p是表示在当前位置之后，P表示在当前位置之前

P 粘贴


yy 拷贝当前行当行于 ddP

4. Undo/Redo

u undo


Ctrl键+r redo

5. 打开/保存/退出/改变文件

:e <path/to/file>打开一个文件


:w存盘


:saveas <path/to/file>另存为


:x， ZZ 或 :wq保存并退出 注:x 表示仅在需要时保存，ZZ不需要输入冒号并回车


:q! 退出不保存 :qa! 强行退出所有的正在编辑的文件，就算别的文件有更改


:bn 和 :bp 你可以同时打开很多文件，使用这两个命令来切换下一个或上一个文件。注：我喜欢使用:n到下一个文件

第三级 更好，更强

更好：重复上一次执行命令

. (小数点) 可以重复上一次的命令


N<command> → 重复某个命令N次 例子： 2dd,3yy

更强：光标跳转

NG 到第 N 行 注意命令中的G是大写的 等同于 : N到第N行


gg 到第一行。注：相当于1G，或 :1


G 到最后一行


按单词移动:


w到下一个单词的开头


e到下一个单词的结尾


匹配括号移动: **程序员非常好用的功能**


%匹配括号移动，包括 (, {, [. 注：你需要把光标先移到括号上


* 和 # 匹配光标当前所在的单词，移动光标到下一个（或上一个）匹配单词（#是上一个,*是下一个）


Ctrl + O向后回退你的光标移动


Ctrl + I向前追赶你的光标移动

第四级 Vim 超能力

**当前行上移动光标: 0 ^ $ f F t T , **

Raw
0 → 到行头
^ → 到本行的第一个非blank字符
$ → 到行尾
g_ → 到本行最后一个不是blank字符的位置。
fa → 到下一个为a的字符处，你也可以fs到下一个为s的字符。
t, → 到逗号前的第一个字符。逗号可以变成其它字符。
3fa → 在当前行查找第三个出现的a。
F 和 T → 和 f 和 t 一样，只不过是相反方向。
`注:  dt 删除所有的内容，直到遇到-- `

VIM在可视模式下的区域选择

#VIM可视模式的区域选择

Raw
vi" → 会选择 foo.
`va"`会选择 "foo".
`vi)`会选择 "foo".
`va)` 会选择("foo").
`v2i)`会选择 map (+) ("foo")
`v2a)` 会选择 (map (+) ("foo"))

VIM在可视模式下的块操作

#VIM在可视模式下的块操作

Raw
(1)`^`到行头
(2)`Ctrl + v` 开始块操作
(3)向下移动 (你也可以使用hjkl来移动光标，或是使用%，或是别的
(4)`I-- [ESC] → I是插入，插入--` , 按`ESC键`来为每一行生效

VIM在insert模式下的自动提示

#VIM在insert模式下的自动提示

在 Insert 模式下，你可以输入一个词的开头，然后按Ctrl + p或是Ctrl + p自动补齐功能就出现了

VIM的宏录制

#VIM的宏录制

qa 把你的操作记录在寄存器 a。
于是 @a 会replay被录制的宏。
@@ 是一个快捷键用来replay最新录制的宏

VIM的可视化选择与自动缩进

#VIM的可视化选择与自动缩进

Raw
`Ctrl +v` 进入可视化模式
`j` 把所有的行连接起来（变成一行）
`< 或 > `左缩进，右缩进
`=`自动缩进  `注：一直没有发现还有这么牛逼的功能funk`

**例子**
Ctrl +v 进入可视化模式
选中相关的行 (可使用 j 或  或是 /pattern 或是 % 等……)
$ 到行最后
A, 输入字符串，按 ESC。

分屏显示

Raw
分屏: :split 和 vsplit
`:split`创建分屏 `注 :vsplit创建垂直分屏`
`Ctrl + 方向键或者hjkl键` 来切换分屏
`Ctrl + w + _    或 Ctrl + w + |`     `注 | 垂直分屏`  最大化尺寸 
`Ctrl 加 + 或者 Ctrl 加 -`   增加尺寸
