匿名函数

关键字lambda表示匿名函数，冒号前面的x表示函数参数

Python
f = lambda x: x * x

闭包

闭包(Closure),关键词:自由变量 和 函数;闭包是由函数和与其引用的自由变量组合而成的实体;

-直白的表述: 函数A(自由变量) 里面嵌套一个函数B(自由变量), 函数B()返回值,函数A返回函数B;

Python
def A(var1):
    def B(var2):
        return "var1:%s;var2:%s" % (var1,var2)
    return B
#调用
obj = A(10)
print obj(10)
print obj(20)

nonlocal

在 python 的函数内可以直接引用外部变量,但不能改写外部变量,因此如果在闭包中直接改写父函数的变量,就会发生错误

在 python2.x 中使用global 语句解决这个问题(不提倡) 在python 3.x 中使用nonlocal 语句解决这个问题

Nonlocal 与 global 的区别在于 nonlocal 语句会去搜寻本地变量与全局变量之间的变量，其会优先寻找层级关系与闭包作用域最近的外部变量

Python
#python 3.x
def cnt(param):
    count = 0 
    def counter():
        nonlocal count
        count += 1
        print("I am",param,"No.",str(count))
    return counter
#调用
db = cnt("doubi")
db()
db()

我们为什么使用闭包? 或者说闭包的作用？

Raw
闭包的使用上有点想类实例化对象:
闭包的最大特点是可以**将父函数的变量与内部函数绑定**，并返回绑定变量后的函数（也即闭包），此时即便生成闭包的环境（父函数）已经释放，闭包仍然存在，这个过程很像类（父函数）生成实例（闭包），不同的是**父函数只在调用时执行，执行完毕后其环境就会释放**，而类则在文件执行时创建，一般程序执行完毕后作用域才释放，因此对一些需要重用的功能且不足以定义为类的行为，使用闭包会比使用类占用更少的资源，且更轻巧灵活;

闭包用于封装,有效的减少了函数参数的数目,这一点和偏函数()很像;
Python
1.闭包
def A(base):
    def int2(arg):
        return int(arg,base)
    return int2
#调用
obj = A(base=2)
print obj("1000000") #return 64

偏函数

Python
2.偏函数
格式:partial(函数, 函数参数=参数值)
from functools import partial
int2 = functools.partial(int, base=2)
int2('1000000') ＃return 64

总结: 偏函数是闭包的一种应用,它让封装函数,减少参数数量很有用

装饰器

装饰器就是一种的闭包的应用,只不过其传递的是函数;

在面向对象（OOP）的设计模式中，decorator被称为装饰模式。OOP的装饰模式需要通过继承和组合来实现，而Python除了能支持OOP的decorator外，直接从语法层次支持decorator。Python的decorator可以用函数实现，也可以用类实现;

@functools.wraps(func) 装饰器的作用是保证经过装饰器装饰的函数的func.name属性不被改变为装饰器中的函数名称

简单装饰器

Python
#-*-coding: utf-8 -*-
'''
Created on 2016年6月28日
@author: daixiang
'''
import functools

def add_handle(fun):
    """装饰器接收函数”””
    @functools.wraps(func)
    def handle():
        res = fun()
        return " ".join([res,"handle"]) 
    return handle
    
@add_handle  #python语法
def make_cake():
    return "cake"
    
if __name__ == '__main__':
    print make_cake()

def add_arg(fun):
    """对带参数的函数进行装饰"""
    @functools.wraps(func)

类装饰器 用init()和call()来实现

简单类装饰器

Python
#-*-coding: utf-8 -*-
'''
Created on 2016年6月28日
@author: daixiang
'''
from functools import wraps

class add_arg_foo(object): 
    """
    带参数的类装饰器：当装饰器有参数的时候，__init__() 成员就不能传入func了，而是在__call__中传入;
    此时在__init__()函数中传递装饰器参数，在__call__中重新定义装饰器,并在wrapped()函数传递函数参数；
    """
    def __init__(self, func):
        self.func = func
        
    def __call__(self):
        print "self.func.__name__:",self.func.__name__
        return " ".join([self.func(), "test"])    

@add_arg_foo
def make_cake():
    return "cake"
    
if __name__ == '__main__':
    print make_cake()

装饰器和被装饰的函数都带参数

Python
#-*-coding: utf-8 -*-
'''
Created on 2016年6月28日
@author: daixiang
'''
from functools import wraps

class add_arg_foo(object): 
    """
    带参数的类装饰器：当装饰器有参数的时候，__init__() 成员就不能传入func了，而是在__call__中传入;
    此时在__init__()函数中传递装饰器参数，在__call__中重新定义装饰器,并在wrapped()函数传递函数参数；
    """
    def __init__(self,parma):
        self.parma = parma

    def __call__(self, func):
        @wraps(func)
        def wrapped(*args,**kwargs):
            print "func.__name__, %s" % func.__name__
            res = func(*args,**kwargs)
            return " ".join([res,self.parma])
        return wrapped
    
@add_arg_foo("handle")
@add_arg_foo("handle")
def make_cake(parma):
    return "cake"
    
if __name__ == '__main__':
    print make_cake("parma")

装饰类

Python
#-*-coding: utf-8 -*-
'''
Created on 2016年6月28日
@author: daixiang
'''
from functools import wraps

class add_arg_foo(object): 
    """
    对类进行装饰，对类进行一系列的操作，最后可以return一个新的类，或者类的实例，或着其他
    """
    def __init__(self,parma):
        self.parma = parma

    def __call__(self, cls):
        @wraps(cls)
        def wrapped(*args,**kwargs):
            print "cls.__name__, %s" % cls.__name__
            res = cls(*args,**kwargs)
            res.add_new_parma = "add_new_parma"
            return res #对类的属性进行增加操作，实例化，最后返回修改后的类的实例
        return wrapped
    
    
@add_arg_foo("handle")
class make_cake(object):
    def __init__(self,parma):
        self.parma = " ".join(["cake",parma])
    
    def get(self):
        return self.parma
    
if __name__ == '__main__':
    obj = make_cake("parma")
    print obj.parma
    print obj.add_new_parma
    print obj.get()

装饰器打日志

Python
from functools import wraps
def logger(fn):
    @wraps(fn)
    def wrapper(*args, **kwargs):
        ts = time.time()
        result = fn(*args, **kwargs)
        te = time.time()
        print "function      = {0}".format(fn.__name__)
        print "    arguments = {0} {1}".format(args, kwargs)
        print "    return    = {0}".format(result)
        print "    time      = %.6f sec" % (te-ts)
        return result
    return wrapper
