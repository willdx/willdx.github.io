1.说明

环境字典fabric.state.env作为全局单例实现,为方便使用fabric将其封装在fabric.api中


env中的键也被称为环境变量,Fabric的大部分行为可以通过修改env(例:env.hosts)控制


env是dict的子类,同时支持属性访问方式.host_string等同于env['host_string']

Raw
属性访问的方式使得代码更简洁:
print("Executing on %s as %s" % (env.host, env.user))等同于print("Executing on %(host)s as %(user)s" % env)

修改环境变量env的值

Raw
(1)可以直接在模块级别定义环境变量env的值
(2)也可以通过上下文管理器进行临时的修改,
(3)在shell命令行使用fab命令加上参数也可以临时设置,只对当前命令有效

2.上下文管理器context_managers

2.1 settings

Bash
from fabric.api import settings, run

    def exists(path):
        with settings(warn_only=True): 
            return run('test -e %s' % path)

3.环境变量完整列表

user: 指明默认运行的用户

Bash
from fabric.api import env, run

    env.user = 'implicit_user'
    env.hosts = ['host1', 'explicit_user@host2', 'host3']
    
    def print_user():
        with hide('running'):
            run('echo "%(user)s"' % env)
