Django学习笔记扩展02_python中new/init/call方法

Posted April 07, 2017

Python
1. __new__(cls): 对象的创建，是一个静态方法，第一个参数是cls;
2. __init__(self): 对象的初始化， 是一个实例方法，第一个参数是self;
3. __call__(self): 类的实例化对象可call，注意不是类，是对象, 第一个参数是self;

注意: 先有创建，才有初始化; 即先__new__，而后__init__;

1. 一个call函数的例子

将类变成一个可调用对象

  • 代码
Python
class Person(object):
    def __init__(self, name, gender):
        self.name = name
        self.gender = gender

    def __call__(self, friend):
        print 'My name is %s...' % self.name
        print 'My friend is %s...' % friend
  • 实例化的对象能被调用(像函数一样)
Python
>>> p = Person('Bob', 'male')
>>> p('Tim')
My name is Bob...
My friend is Tim...

单看 p('Tim') 你无法确定 p 是一个函数还是一个类实例, 在类中使用call方法能够让类实例像函数一样调用;
