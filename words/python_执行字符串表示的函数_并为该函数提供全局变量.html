python_执行字符串表示的函数_并为该函数提供全局变量

Posted April 07, 2017

Bash
#!usr/bin/env python
    #coding:utf-8
    namespace = {'name':'wupeiqi','data':[18,73,84]} ＃数据字典
    code =  '''def hello() return "name %s,age %d" %(name,data[0],) ''' #字符串函数
    func = compile(code, '<string>', "exec") ＃将字符串式函数转化为真正函数 
    exec func in namespace ＃将函数 hello 添加到namespace字典中（key为hello），同时也将python的所有内置函数添加到namespace字段中（key为__builtins__）
    result = namespace['hello']()  #执行Hello函数并将返回值复制给result
    print result #输入result

上述代码解析：把字符串变成了函数并且还为该函数提供了全局变量

Raw
namespace中的内容好比是一个个的全局变量，即 
name ＝ wupeiqi
data ＝ [18,73,84]
def hellocute():
    return  "name %s ,age %d" %(name,data[0],)

对于该功能，它就是python的web框架中模板语言部分至关重要的部分，因为在模板处理过程中，首先会读取html文件，然后分割html文件，再然后讲分割的文件组成一个字符串表示的函数，再再然后就是利用上述方法执行字符串表示的函数。
